float ** Geotiff::GetRasterBand ( int  z)

This function reads a band from a geotiff at a specified vertical level.

(z value, 1..n bands). To this end, the Geotiff's GDAL data type is passed to a switch statement, and the template function GetArray2D (see below) is called with the appropriate C++ data type. The GetArray2D function uses the passed-in C++ data type to properly read the band data from the Geotiff, cast the data to float**, and return it to this function. This function returns that float** pointer.

z
Returns
